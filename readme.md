## Plugins incluidos

* Advanced Custom Fields
* Contact Form 7
* Flamingo
* Duplicate post
* Post Types Order e Taxonomy Terms Order
* Simple History
* Woocommerce
* Ultimate Member
* Favorites
* WP Mail SMTP

## Tecnologias do tema
* Sass
* Bootstrap
* jQuery
* PHP
* Laravel-Mix

## Instalação do template
1. Clone o projeto na pasta htdocs 

2. Para acessar o código do tema, `wp-content/themes/mercadin`
